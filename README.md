# My Streamlit App

This is a Streamlit application that showcases various skills in Data Science, Machine Learning, and other relevant tools. The app is designed to provide an interactive experience similar to the original website.

## Project Structure

```
my-streamlit-app
├── src
│   ├── app.py          # Main entry point of the Streamlit application
│   └── assets
│       └── styles.css  # Custom styles for the application
├── requirements.txt     # List of dependencies
└── README.md            # Project documentation
```

## Requirements

To run this application, you need to have Python installed along with the required libraries. You can install the necessary dependencies using the following command:

```
pip install -r requirements.txt
```

## Running the Application

To start the Streamlit application, navigate to the `src` directory and run:

```
streamlit run app.py
```

This will launch the application in your default web browser.

## Features

- Displays skills in Data Science and Machine Learning.
- Custom styling to enhance the user interface.
